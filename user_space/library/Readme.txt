Steps for compilation
1. make
To genrate:
	a. Static Library and its test program
	b. Dynamic Library and its test program
	c. Static and dynamic library in muti folders and a single program to test both

2. make staticlib
To genrate static library and its test program

3. make dynamiclib
To genrate dynamic library and its test program

4. make multilib
To genrate static and dynamic lib in multiple folder

Steps to run program
Go to perticular directory e.g static_lib and run
./test_prog
